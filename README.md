# Installation and use

## Dependency
- PHP >= 7.1.3
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
- Ctype PHP Extension
- JSON PHP Extension

```
 -clone this repo
$ cd project_directory
$ cp .env.example .env
$ composer install
$ php artisan key:generate
$ php artisan serve
```
Now visit and login: http://localhost:8000 \
